Desarrollo WEB

es un término que define la creación de sitios web para Internet o una intranet. Para conseguirlo se hace uso de tecnologías de software del lado del servidor y del cliente que involucran una combinación de procesos de base de datos con el uso de un navegador web a fin de realizar determinadas tareas o mostrar información

FRONT END

Frontend es la parte de un sitio web que interactúa con los usuarios, por eso decimos que está del lado del cliente.

  • HTML: Es el código que se utiliza para estructurar y desplegar una página web y sus contenidos
  • CSS: Es un lenguaje que define la apariencia de un documento escrito en un lenguaje de marcado
  • JavaScript: s un lenguaje de secuencias de comandos que te permite crear contenido de actualización dinámica, controlar multimedia, animar
